# ВСУ РУ19А-300 (Ан-24)

## Общее описание

**РУ19А-300** — вспомогательная силовая установка на самолете Ан-24. Выполняет три функции:
1. **На земле (ВСУ)**: питает бортсеть 27В через ГС-24
2. **При запуске**: помогает раскручивать основные двигатели АИ-24
3. **В воздухе (бустер)**: реактивный двигатель с тягой 900 кгс

---

## Компоненты

### 1. ГС-24 (Стартер-Генератор)

Электрическая машина двойного назначения:

**Режим мотора (стартер):**
- Потребляет ток от аккумуляторов
- Раскручивает вал ВСУ
- Ток падает с ростом оборотов (противо-ЭДС)

**Режим генератора:**
- После 45% RPM переключается на генерацию
- Выдает ток в бортовую сеть
- Напряжение стабилизируется регулятором РУГ-82

**Параметры:**
- Max RPM: 16000
- Idle RPM: ~60% (9600)
- Internal resistance: 0.025 Ohm (мотор)
- Norton resistance: 0.08 Ohm (генератор)
- Max ток генератора: 400А (ограничен 100А для стабильности)

### 2. РУГ-82 (Регулятор напряжения)

Угольный регулятор - поддерживает 28.5В на шине.

**Интерфейс:**
- Вход: `v_gen` — напряжение генератора (мониторинг)
- Выход: `k_mod` — коэффициент возбуждения (0...1)

**Алгоритм:**
```
error = v_target - v_gen
k_mod += kp * error * dt

Если V > 28.5 → уменьшить k_mod (меньше возбуждения)
Если V < 28.5 → увеличить k_mod (больше возбуждения)
```

### 3. Автомат пуска (ПСГ-15)

Управляет последовательностью запуска ВСУ.

**Временная диаграмма:**

| Время (сек) | Действие |
|-------------|----------|
| 0 | Включение стартера |
| 2 | Включение зажигания (свечи) |
| 7 | Открытие топливного клапана |
| ~12 | Проверка "подхвата" |
| 24-30 | Отключение стартера |

## State Machine (Конечный автомат)

```
[OFF] → [CRANKING] → [IGNITION] → [RUNNING]
                ↑              ↓
                └──────────────┘ (если timeout)
                        ↓
                   [STOPPING] → [OFF]
```

### Состояния:

**OFF**: Ожидание. ВСУ не работает.

**CRANKING** (0-2 сек):
- Стартер раскручивает ротор
- RPM: 0 → 2000 (12.5%)
- Температура T4: медленно растет

**IGNITION** (2-5 сек):
- Подано топливо, включено зажигание
- Турбина помогает стартеру
- RPM: 2000 → 5000 (12.5% → 31%)
- **Критическая зона**: если RPM < 40%, возможен "жаркий старт"

**RUNNING** (5+ сек):
- ВСУ работает самостоятельно
- RPM: 5000 → 9600 (31% → 60% idle)
- T4 стабилизируется

**STOPPING**:
- Аварийная остановка
- Охлаждение T4

---

## Thermal Model (Модель температуры)

### T4 — Температура газов за турбиной

**Физика:**
- Нагрев: от сгорания топлива
- Охлаждение: поток воздуха через компрессор (зависит от RPM²)

**Формула:**
```
dT = (heating - cooling + thermal_stress) * dt

heating = 120 °C/сек (при ignition)
cooling = 110 * (RPM/60%)²
thermal_stress = (0.4 - RPM) * 50 (только при RPM < 40%)
```

**Параметры:**
- Ambient: 20°C
- Target T4 (idle): 400°C
- Max safe T4: 750°C (аварийный останов)

### "Жаркий старт" (Hot Start)

**Сценарий:** слабые аккумуляторы → низкое напряжение → медленный разворот RPM

**Проблема:**
1. Напряжение на шине падает до 14-16В
2. Стартер не может быстро раскрутить ротор
3. При RPM < 40%: thermal_stress > 0
4. T4 растет быстрее чем охлаждение
5. При T4 > 750°C → аварийный останов

**Решение в модели:**
```
Если T4 > t4_max (750°C):
    state = STOPPING
    "HOT START ABORT!"
```

## Электрическая модель (Нортон)

### Режим мотора:
```
I = (V_bus - K_motor * RPM) / R_internal
```

### Режим генератора:
```
I_no = I_max * Phi(RPM) * k_mod

Phi(RPM):
    - 0% ... 40%: 0 (генератор не работает)
    - 40% ... 60%: линейный подъем (0 → 1)
    - 60%+: 1 (полная мощность)
```

### Важные исправления (баги и решения)

#### 1. Back-EMF слишком большой
**Проблема**: Изначально использовали `back_emf = 0.5 * current_rpm`. При 16000 RPM это давало 8000V - в 300 раз больше батареи!

**Решение**: `back_emf = 38.0 * rpm_percent` - при 100% RPM получаем 38V, что соответствует реальной машине.

```cpp
constexpr float K_MOTOR_BACK_EMF = 38.0f;
float rpm_percent = current_rpm / target_rpm;
float back_emf = K_MOTOR_BACK_EMF * rpm_percent;
```

#### 2. DMR-400 зависание при запуске
**Проблема**: Круговая зависимость - DMR не закрывается пока генератор не раскрутится, но стартер не получает питание пока DMR закрыт.

**Решение**: Напряжение-based detection:
- DMR закрывается когда `V_gen > 20V` (генератор уже работает)
- Во время CRANKING генератор потребляет, напряжение низкое → DMR остаётся открытым
- После перехода в RUNNING генератор даёт 28V → DMR закрывается

#### 3. Подключение стартера в обход DMR
**Физика**: В Ан-24 пусковой ток идёт напрямую от аккумуляторов, а не через ДМР.

**Реализация**: Два порта в RU19A:
- `v_start` - питание стартера (подключается к батарее напрямую)
- `v_bus` - выход генератора (идёт через ДМР на шину)

#### 4. Реалистичная просадка напряжения
При запуске ток 700-850A вызывает просадку до 17-21V (вместо 28V на батарее).

```cpp
// При V_start=24V, back_emf=0 (старт):
i_consumed = (24 - 0) / 0.025 = 960A
// Это даёт просадку на внутреннем сопротивлении АКБ
```

## DMR-400 Логика

```
В СОСТОЯНИИ OFF/CRANKING:
  V_gen (на генераторе) < 20V  →  DMR открыт (не подключён к шине)

ПРИ ПЕРЕХОДЕ В RUNNING:
  Генератор даёт 28-30V  →  DMR проверяет условие:
  IF V_gen > V_bus + 2.0V AND V_gen > 20V:
      DMR закрывается (контактор замыкается)
      Генератор подключается к шине

ЕСЛИ ОБРАТНЫЙ ТОК (батарея заряжает генератор):
  IF V_bus > V_gen + 10V:
      DMR размыкается (защита от заряда генератора)
```
